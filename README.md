# ESLint Config Base

## JavaScript

1. Install the correct version of package, which are listed by the command:

```bash
npm install "@sanv/eslint-config-base" --save-dev
```

2. Add next line to your `.eslintrc.js`:

```json
{
  "extends": "@sanv/eslint-config-base"
}
```
